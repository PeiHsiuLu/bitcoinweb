# bitcoinweb
 網際網路概論專題 - 比特雪球DCA網站

**開啟方式**  
下載檔案以後，找到 client 資料夾中的 src 檔案，並在以上檔案的終端機輸入以下指令：  
```bash
npm install
npm start
```
接著，找到 server 端的 index 在server端的終端機輸入以下指令：  
```bash
node index.js
```
